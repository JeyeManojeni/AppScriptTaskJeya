# Appscrip Task - Product Listing Page

A responsive Product Listing Page built with React.js, featuring filters, API integration, and SEO optimizations.

## Setup
1. Clone the repo: `git clone <your-repo-url>`
2. Install dependencies: `npm install`
3. Run locally: `npm start`
4. Build for production: `npm run build`
5. Deploy to Netlify by uploading the `build` folder.

## Features
- Responsive design (desktop, tablet, mobile).
- Filters for category and price.
- API data from Fake Store API with fallback.
- SEO-friendly with semantic HTML and schema.org.

## Technologies
- React.js
- CSS (Grid/Flexbox)
- Axios for API calls
